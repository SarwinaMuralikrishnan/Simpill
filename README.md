# 🏥 SimPill – AI Powered Smart Healthcare & Medicine Management System

## 📌 Overview

SimPill is an AI-powered healthcare and medicine management system designed to simplify hospital workflows and improve patient care. The platform integrates multiple healthcare stakeholders such as administrators, doctors, receptionists, pharmacists, lab technicians, cashiers, patients, and caregivers into a single system.

The system provides features including patient management, AI-assisted prescription generation, medicine reminders, billing, pharmacy management, lab management, relative access, and reward-based medication adherence tracking.

---

## ✨ Key Features

### 🔐 Authentication & Role Management

* Secure user authentication
* Password hashing and session management

* Role-based dashboards

* Patient profile creation

### 🤖 AI Prescription Generator

* AI-assisted medicine suggestions
* Editable prescriptions
* Prescription confirmation workflow

### 💊 Pharmacy Management

* Prescription dispensing

### 💰 Billing System

### ⏰ Medicine Reminder System

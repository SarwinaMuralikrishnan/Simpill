SimPill AI is a modern AI-powered hospital management and patient care ecosystem designed to simplify healthcare workflows for patients, doctors, caregivers, nurses, pharmacies, and hospital administrators.

The platform combines hospital management features with artificial intelligence to provide smart healthcare assistance, medicine adherence tracking, AI prescription scanning, appointment management, telemedicine, emergency support, and healthcare analytics.

Installation
Clone Repository
git clone <your-repository-link>
cd simpill_ai
Create Virtual Environment
python -m venv venv
Activate Environment
Windows
venv\Scripts\activate
Linux / Mac
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Run Application
python run.py
